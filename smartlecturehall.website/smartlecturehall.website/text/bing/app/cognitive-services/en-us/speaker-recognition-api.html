Speaker Verification

Use your voice for authentication. The API can be used to power applications with an intelligent verification tool. If the speaker claims to be of a certain identity use voice to verify this claim. To see how is works, select a pass phrase from the given list of phrases. Use that phrase and record/upload three audio samples to register your voice with the service, this step is called "enrollment". After your enrollment is completed, you can start the verification step using a different voice recording or phrase to test the service. When you use the speaker demonstration applications you consent to providing your voice input data to Microsoft for service improvement purposes.

Read the phrase above or upload an audio file reading it to enroll your voice.

                                            Speaker Identification
                                        


                                        

                                            Identify who is speaking. The API can be used to determine the identity of an unknown speaker. Input audio of the unknown speaker is paired against a group of selected speakers, and in the case there is a match found, the speakerâ€™s identity is returned.

We have selected 5 different US presidents and enrolled them to the service using one of the speeches they gave. To see how the demo works, select a speech for one of the presidents by clicking on the sample audios below, or upload one of your own, to test how to automatically identify which president is speaking.

When you use the speaker demonstration applications you consent to providing your voice input data to Microsoft for service improvement purposes.
